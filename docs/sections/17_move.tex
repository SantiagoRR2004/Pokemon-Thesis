\section{\lang{Hyperparameter}{Hiperparámetro} \emph{move}} \label{anexo:hyperMove}

\ifthenelse{\equal{\LANG}{es}}{
    El hiperparámetro \emph{move} define qué representación se usará cada movimiento. Este, junto con los hiperparámetros \emph{player} y \emph{pokemon}, determinan el número de parámetros que tiene la primera capa de las redes neuronales (observación). La tabla \ref{tab:moveParameters} muestra las $11$ clases de representación. Las representaciones se nombran igual que la implementación de sus correspondientes clases en el código y siguen la nomenclatura de \emph{MoveXX}, donde \emph{XX} representa el nivel de complejidad. La clase con número superior engloba a la clase que tiene un número inferior, es decir, tiene todos los atributos de la clase inferior y algunos más. \emph{S} significa que es la clase más compleja y que intenta representar todo sin que haya límite en el número de parámetros.
}{}
\ifthenelse{\equal{\LANG}{en}}{
    The \emph{move} hyperparameter defines which representation will be used for each move. Together with the \emph{player} and \emph{pokemon} hyperparameters, it determines the number of parameters in the first layer of the neural networks (observation). Table \ref{tab:moveParameters} shows the $11$ representation classes. The representations are named in the same way as the implementation of their corresponding classes in the code and follow the \emph{MoveXX} nomenclature, where \emph{XX} denotes the complexity level. A higher-numbered class encompasses the class with the lower number, that is, it contains all the attributes of the lower class and some additional ones. \emph{S} denotes the most complex class, which attempts to represent everything without any limit on the number of parameters.
}{}

\subsection*{Move00}
\endgroup

\ifthenelse{\equal{\LANG}{es}}{
    En el formato \emph{gen9randombattle} hay $950$ movimientos posibles. El único atributo de la clase \emph{Move00} es el índice del movimiento de una lista fija. Un factor a tener en cuenta es que, mientras se van añadiendo más atributos, estos de forma indirecta, limitan el número de movimientos posibles.
}{}
\ifthenelse{\equal{\LANG}{en}}{
    In the \emph{gen9randombattle} format there are $950$ possible moves. The only attribute in class \emph{Move00} is the move index from a fixed list. One point to keep in mind is that, as more attributes are added, they indirectly restrict the number of possible moves.
}{}

\section{\lang{Hyperparameter}{Hiperparámetro} \emph{pokemon}} \label{anexo:hyperPokemon}

\ifthenelse{\equal{\LANG}{es}}{
    El hiperparámetro \emph{pokemon} define qué representación se usará cada Pokémon (no incluye los movimientos). Este, junto con los hiperparámetros \emph{player} y \emph{move}, determinan el número de parámetros que tiene la primera capa de las redes neuronales (observación). La tabla \ref{tab:pokemonParameters} muestra las $12$ clases de representación. Las representaciones se nombran igual que la implementación de sus correspondientes clases en el código y siguen la nomenclatura de \emph{PokemonXX}, donde \emph{XX} representa el nivel de complejidad. La clase con número superior engloba a la clase que tiene un número inferior, es decir, tiene todos los atributos de la clase inferior y algunos más. \emph{S} significa que es la clase más compleja y que intenta representar todo sin que haya límite en el número de parámetros.
}{}
\ifthenelse{\equal{\LANG}{en}}{
    The \emph{pokemon} hyperparameter defines which representation will be used for each Pokémon (moves are not included). Together with the \emph{player} and \emph{move} hyperparameters, it determines the number of parameters in the first layer of the neural networks (observation). Table \ref{tab:pokemonParameters} shows the 12 representation classes. The representations are named in the same way as the implementation of their corresponding classes in the code and follow the \emph{PokemonXX} nomenclature, where \emph{XX} denotes the complexity level. A higher-numbered class encompasses the class with the lower number, that is, it contains all the attributes of the lower class and some additional ones. \emph{S} denotes the most complex class, which attempts to represent everything without any limit on the number of parameters.
}{}

\subsection*{Pokemon00}
\endgroup

\ifthenelse{\equal{\LANG}{es}}{
    Esto ocurrirá para todas las representaciones de un Pokémon; en su interior, va a tener el vector que representa a cada uno de los $4$ movimientos que se obtiene de la clase que representa el hiperparámetro \emph{move}. En una misma batalla, el orden de estos no va a cambiar y se va a ir añadiendo más información. Si todavía no se conoce nada de un movimiento, se usa un vector de ceros.
}{}
\ifthenelse{\equal{\LANG}{en}}{
    This applies to every Pokémon representation; internally, it contains the vector representing each of the $4$ moves obtained from the class that implements the \emph{move} hyperparameter. Within a single battle, their order does not change, and more information is added progressively. If nothing is known about a move yet, a zero vector is used.
}{}

\ifthenelse{\equal{\LANG}{es}}{
    En el formato \emph{gen9randombattle} hay $1329$ formas posibles de Pokémon. El único atributo que la clase \emph{Pokemon00} es el índice de la forma del Pokémon de una lista fija. Un factor a tener en cuenta es que, mientras se van añadiendo más atributos, estos de forma indirecta, limitan el número de formas posibles de Pokémon.
}{}
\ifthenelse{\equal{\LANG}{en}}{
    In the \emph{gen9randombattle} format there are $1329$ possible Pokémon forms. The only attribute in class \emph{Pokemon00} is the index of the Pokémon's form from a fixed list. One point to keep in mind is that, as more attributes are added, they indirectly limit the number of possible Pokémon forms.
}{}

\subsection*{Pokemon02}
\endgroup

\ifthenelse{\equal{\LANG}{es}}{
    Esta representación añade $3$ enteros:
}{}
\ifthenelse{\equal{\LANG}{en}}{
    This representation adds $3$ integers:
}{}

\begin{itemize}
    \item
          \ifthenelse{\equal{\LANG}{es}}{
              El índice de la habilidad del Pokémon. Es un número entero que va de 0 a $309$, que es el número de habilidades en el formato \emph{gen9randombattle}.
          }{}
          \ifthenelse{\equal{\LANG}{en}}{
              The index of the Pokémon's ability. It is an integer from 0 to $309$, which is the number of abilities in the \emph{gen9randombattle} format.
          }{}
    \item
          \ifthenelse{\equal{\LANG}{es}}{
              La proporción de vida restante del Pokémon. Es un flotante que va de 0 a 1, donde 0 significa que el Pokémon está debilitado y 1 significa que tiene toda la vida.
          }{}
          \ifthenelse{\equal{\LANG}{en}}{
              The Pokémon's remaining HP ratio. It is a float from 0 to 1, where 0 means the Pokémon has fainted and 1 means it is at full health.
          }{}
    \item
          \ifthenelse{\equal{\LANG}{es}}{
              El índice del objeto que tiene el Pokémon. Es un número entero que va de 0 a $533$, que es el número de objetos en el formato \emph{gen9randombattle}.
          }{}
          \ifthenelse{\equal{\LANG}{en}}{
              The index of the item held by the Pokémon. It is an integer from 0 to $533$, which is the number of items in the \emph{gen9randombattle} format.
          }{}
\end{itemize}

}{}

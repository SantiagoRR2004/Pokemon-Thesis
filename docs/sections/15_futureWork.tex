\section{Vías de trabajo futuro}

A la arquitectura de las redes. Se podría usar alguna técnica de RNN para que la red pueda tener más información durante la batalla e igual intentar anticipar planes a largo plazo del oponente. También se pueden intentar usar técnicas más complejas como atención.

Para la representación del estado se podría entrenar un autoencoder que compactase la representación de cada Pokémon y así reducir el número de parámetros de la representación del estado global.

Usar técnicas más avanzadas que REINFORCE o Actor-Critic, como PPO o DDPG.

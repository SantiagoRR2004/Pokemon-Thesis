\section{Modelado completo} \label{anexo:modeling}

El documento utilizado para apoyar la formulación del marco teórico corresponde a un borrador interno de trabajo elaborado por David Olivieri dentro de una línea de investigación en curso sobre \emph{graphon world models} y aproximaciones de tipo \emph{graphon mean-field} para aprendizaje por refuerzo. Este trabajo forma parte de un proyecto más amplio, todavía no publicado, que estudia cómo representar entornos y juegos de RL mediante kernels tipo graphon, operadores de control y procedimientos de \emph{coarse-graining}, con el objetivo de analizar familias de juegos, estructuras colectivas y posibles clases universales de comportamiento.
